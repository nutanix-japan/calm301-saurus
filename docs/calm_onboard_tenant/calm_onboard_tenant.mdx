# Calm: Onboard Tenant

## Pre-requsities:

1.  In Prism Central, Micro-service infrastructure was enabled
2.  In Prism Central, Advanced Network was enabled
3.  PC2022.1.02
4.  Calm 3.5.0

## Overview

The service provider would like to onboard each tenant.

![](images/vpc-overview.png)

### Create a blueprint to onboard tenant

1.  Click on **Create Blueprint**. Create a **Multi VM/Pod** blueprint.

> ![](images/create-blueprint.png)

1.  Select the **Credentials** tab. Create a new credential for **PC
    Credential** and enter desired User/Pass\*\*

> ![](images/pc_cred.png)

1.  Create the following variables on the right side of the screen.

> -   vpc_name Default value: empty. **Mark this variable mandatory**
> -   external_routable_ip Default value: **Follow the cluster. Eg
>     10.x.x.0/24**. **Mark this variable mandatory**
> -   dns Default value: **1.1.1.1**. **Mark this variable mandatory**
> -   PC_IP Default value: **Follow the cluster PC. Eg 10.x.x.39**
>
> ![](images/variables.png)

1.  Create the following dynamic variable: **external_subnet** on the
    right side of the screen. **Replace the password with the Prism
    Central password.**

``` python
user = "@@{PC Credential.username}@@"
password = "@@{PC Credential.secret}@@"
ip = "@@{PC_IP}@@"

headers = {'Accept': 'application/json', 'Content-Type': 'application/json'}
url_method = "POST"

payload = {
  "kind": "tunnel",
  "filter": "name==NTNX_LOCAL_AZ_VPC_@@{vpc_name}@@_Tunnel"
}

base_url = "https://" + ip + ":9440/api/nutanix/v3/tunnels/list"
url = base_url
r = urlreq(url, url_method, auth="BASIC", user=user, passwd=password, params=json.dumps(payload), verify=False, headers=headers)
print "Status code: {}".format(r.status_code)
print "Output: {}".format(r.text)
if r.ok:
  print "Successful retrieval of tunnel VM"
  tunnel_json = r.json()
  for tunnel in tunnel_json['entities']:
    if tunnel['metadata']['name'] == 'NTNX_LOCAL_AZ_VPC_@@{vpc_name}@@_Tunnel':
      print "tunnel_uuid={}".format(tunnel['metadata']['uuid'])
      exit(0)
else:
  exit(1)
```
